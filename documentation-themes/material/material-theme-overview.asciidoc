== Getting started
++++

<h3 demo-section>1. Install</h3>
<p>To start using the Material theme you need to install the
  <a href="https://github.com/vaadin/vaadin-material-styles">vaadin-material-styles</a> package using
  <a href="https://www.npmjs.com/">npm</a>.</p>

++++
```
npm i @vaadin/vaadin-material-styles
```
++++

<h3 demo-section>2. Import and include main style modules</h3>
<p>Import and include the main style modules in the global scope:</p>
++++
[source,html]
----
<!-- Place these in your index.html -->
<script type="module">
  import '@vaadin/vaadin-material-styles/color.js';
  import '@vaadin/vaadin-material-styles/typography.js';
</script>

<!-- Include the main style modules in global scope (index.html) -->
<custom-style>
  <style include="material-color-light material-typography"></style>
</custom-style>
----
[NOTE]
See link:https://polymer-library.polymer-project.org/3.0/api/elements/custom-style[documentation for the <custom-style> element]
